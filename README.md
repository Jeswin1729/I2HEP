# Internship Week 1: Personal Portfolio Website

## Project Overview
This project is a personal portfolio website developed as part of the Week 1 assignment for I2HEP. The objective was to establish a foundational understanding of web structure and design using semantic HTML5,CSS and Javascript.

## Features
- **Navigation:** A navigation bar for consistent access to Home, Projects, and Contact pages.
- **Home Page:** Professional introduction and academic background as a 3rd-year B.Tech student in AI & Data Science.
- **Projects Page:** Documentation of Academic projects done previously.
- **Responsive Layout:** Basic styling implemented to ensure readability across different screen sizes.

## Technical Stack
- **Languages:** HTML5, CSS3
- **Development Environment:** VS Code
- **Version Control:** Git & GitHub

# Requirements:
### Assignment: 
Build a responsive personal portfolio website with at least 3 pages (Home, Projects, 
Contact). Mobile-first design, navigation bar, contact form (dummy submit), hosted 
on GitHub Pages. 

### Requirements: 
- Add a navigation bar with smooth scrolling and highlight for the active section. 
- Projects page must showcase at least 3 projects with images, descriptions, and 
links. 
- Contact form with JavaScript validation (required fields, email format) and 
dummy submission. 
- Mobile-first responsive design tested on desktop, tablet, and mobile. 
- Deliverables: GitHub repository + live deployment (GitHub Pages/Netlify).
